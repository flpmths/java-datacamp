Java possui diferentes types
String, int, double, short, long, byte

String = texto
int = numero inteiro
double = numero com casa decimal
float = numero com casa decimal porem sem precisao
short = -32,768 to 32,767
long = -2^63 and 2^63
byte = -128 to 127
char = alphabet, punc, numbers, etc - caractere unico
boolean = true OR false

Para criar uma variavel String = "use aspas duplas"

String.length()
.toUpperCase()
.toLowerCase()

e possivel declarar uma variavel e atribuir valor depois

// Declare now
String testMessage;

// Assign later
testMessage = "This is a test";

Metodo para retornar o comprimento da string (numero de caracteres)
int msgLength = testMessage.length();

Metodo para converter String para LOWER Case
String lowerMessage = testMessage.toLowerCase();

Metodo para converter String para UPPER Case
String upperMessage = testMessage.toUpperCase();


length nao precisa de parenteses dentro de um array, trata-se de um atributo e nao um metodo !!
